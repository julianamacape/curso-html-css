Trabalhando com listas

Listas ordenadas

Para criar listas ordenadas (com números) utilizamos a tag <ol> e dentro dela utilizamos a tag <li>, que é a tag que se encarrega de organizar a numeração.
Veja o exemplo abaixo:

  1. 7h30 levantar
  2. 8h estudar
  3. 10h iniciar trabalho
  4. 15h aula Teens
  5. 16h30 aula regular (tarde)
  6. 18h45 aula regular (noite)

Digamos que eu tenha esquecido de inserir a atividade "13h almoçar". Basta incluir essa atividade (dentro da tag <ol> já aberta) utilizando a tag <li>.
Você verá que a própria tag <li> se encarrega de atualizar a numeração. Vai ficar assim:

  1. 7h30 levantar
  2. 8h estudar
  3. 10h iniciar trabalho
  4. 13h almoçar
  5. 15h aula Teens
  6. 16h30 aula regular (tarde)
  7. 18h45 aula regular (noite)

É possível escolher outros tipos de numeradores. Para isso, utilizamos o parâmetro <type> dentro da tag <ol> de abertura:
<ol type="nmr">

Listas Não Ordenadas

Essas listas usam marcadores que não indicam uma sequência. Usamos a tag <ul> e dentro dela usamos a tag <li> para indicar os itens da lista.

  • 7h30 levantar
  • 8h estudar
  • 10h iniciar trabalho
  • 13h almoçar
  • 15h aula Teens
  • 16h30 aula regular (tarde)
  • 18h45 aula regular (noite)
Também é possível mudar a formatação do marcador usando o parâmetro "type" dentro da tag <ul> assim: <ul type="...">
