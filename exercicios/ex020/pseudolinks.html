Pseudo-classes aplicadas a links

  • Padrão
    •     
        a {
            color: blue;
            }
  • Links Não-Visitados
    •     
        a:link {
            color: blue;
            }
  • Links Visitados
    
        a:visited {
            color: purple;
            }
  • Quando passa o mouse por cima 
    •     
        a:hover {
            color: red; 
            }
  • Link em foco
    •     
        a:focus {
            color: orange;
            }
  • Link ativo
    •     
        a:active {
            color: green;
            }
