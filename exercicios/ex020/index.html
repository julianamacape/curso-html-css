Pseudo-Class

As Pseudo-Class devem estar relacionadas a um elemento ou uma classe e elas dizem respeito ao estado de um elemento ou de uma classe. Veja o exemplo abaixo: 

        
            <style>
                h1:hover {
                    color: red;
                }
            </style>

O sinal de : (dois pontos) é o que usamos para indicar uma pseudo-class. Escolhemos o estado "hover", mas há várias opções. "Hover" é a ação que o usuário deve tomar para que a pseudo-class funcione. Nesse caso, a ação é "passar o mouse por cima" e o que acontecerá está declarado logo abaixo, no caso, a cor da fonte do título h1 vai ficar vermelho.
