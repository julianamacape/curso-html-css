Pseudo-Class e Pseudo-Element

Pseudo-Element

O pseudo-element mexe no conteúdo periférico do elemento. O símbolo do pseudo-element é ::
Como exemplo, vamos utilizar a configuração "::after" (depois do elemento) e "::before" (antes do elemento). Lá em CSS fica assim: 

        
            <>style>
                a::after {
                    content: '[LINK]';
                }
        
                a::before {
                    content: 'Acesse';
                }
            </style>

O que isso quer dizer é que antes de todo link haverá o conteúdo 'Acesse' e que depois de todo link haverá o conteúdo '[LINK]'.
