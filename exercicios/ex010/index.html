Usando links externos

Para disponibilizar links externos utilizamos a tag <a>. Dentro do parâmetro "href=" nós inserimos a URL completa da página que será o destino do usuário quando ele clicar no seu link. E dentro dos sinais > < devemos inserir o trecho do texto que representará o link. 

        <a href="aqui dentro a URL completa">aqui dentro o trecho do texto que carrega o link</a>

Exemplo: Você pode acessar o meu repositório público no GitHub clicando AQUI.

Perceba que ao clicar no link "AQUI" o usuário é levado a outra página e o navegador utiliza-se da mesma aba já aberta, ou seja, o site em que o usuário estava fica pra trás. Se você quiser que o link leve o usuário para o site de destino abrindo uma outra aba, então utilize os parâmetros target="_blank" rel="external"

Exemplo: Você pode acessar o meu repositório público no GitHub clicando AQUI.

Links Internos

No caso de você criar links cujas páginas de destino são dentro do seu próprio site, criaremos links internos.

Links para Download

Para criar um link para download devemos utilizar a tag <a> e dentro do parâmetro href="#" inserimos o arquivo que será disponibilizado para download e adicionamos os parâmetros "download" e "type". 

            Baixe o aquivo <a href="file.pdf" download="file.pdf" type="application/pdf">aqui!</a>
