A regra da w3schools é clara: dentro de um documento html, uma 'id' só pode ser usada uma única vez. Pode haver outras 'id', mas cada uma delas só pode usar uma única vez.

Quando você quiser aplicar uma determinada formatação de uma 'id' a mais de um elemento, então não é a 'id' que você vai usar; é a 'class'. Veja os exemplos abaixo:
