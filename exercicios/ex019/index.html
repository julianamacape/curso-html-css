Seletores Personalizados

Caso você queira fazer uma declaração personalizada para um determinado seletor, usamos o parâmetro 'id' dentro de uma linha dentro do documento de html.

Título Teste

Perceba que esse <h1> "Título teste" tem uma formatação diferente do <h1> lá do topo da página. Em tese, ao configuar o seletor <h1> em CSS, todos os títulos <h1> receberiam a mesma formatação. E é assim de fato. Usamos a 'id' em CSS justamente para abrir uma exceção. No texto de html, tivemos que fazer o seguinte: 

        
            <h1 id="personalizado">Título Teste</h1>

A denominação 'personalizado' foi da minha cabeça. Escolha o termo que quiser. PREFERENCIALMENTE em termo que destaque a função dessa 'id'.

E lá em CSS, ficou assim: 

        
            <style>
                @import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');
        
                h1#personalizado {
                    font-family: 'Press Start 2P', Arial, Helvetica, sans-serif;
                }
            </style>

Importante mencionar que escrever o nome do seletor enquanto declarando uma 'id' no arquivo de CSS é opcional. Então também poderia ser: 

        
            <style>
                @import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');
        
                #personalizado {
                    font-family: 'Press Start 2P', Arial, Helvetica, sans-serif;
                }
            </style>
