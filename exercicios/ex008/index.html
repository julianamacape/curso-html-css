Texto grande e texto pequeno

Utilizamos a tag <big> para texto grande e a tag <small> para texto pequeno.
Vale a pena salientar que a tag <big> está obsoleta e já aparece na cor vermelha dentro do Visual Studio Code.

Texto deletado (tachado ou strikethrough)

Podemos marcar um texto como excluído, para indicar que ele não deve ser considerado.

Texto inserido (sublinhado ou underlined)

Podemos marcar um texto inserido para indicar que ele foi adicionado depois.
Também existe a tag <u> que é para "underline", mas essa é uma tag não-semântica e a qualquer momento pode ser considerada como obsoleta.

Texto sobrescrito

Utilizamos a tag <sup> e seu texto fica assim. É bom para equações, tipo x2 + 3 = 0

Texto subescrito

Utilizamos a tag <sub> e seu texto fica assim. É bom para fórmulas químicas, tipo H20.
