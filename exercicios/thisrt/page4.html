position: relative

I moved the yellow box relative to its current position by 20px from the left and top without changing the layout around it. When you set the position relative to an element, without adding any other positioning attributes (top, bottom, right, left) nothing will happen. When you add an additional position, such as 'left: 20px;' the element will move 20px to the right and if you add 'top: 20px;' the element will move 20px down from its normal position. Here, you can see that this element is relative to itself. When the element moves, no other element on the layout will be affected.

position: absolute

The positioning is done relative to the first relatively (or absolutely) positioned parent element. The first parent element MUST HAVE the 'position' attribute set in CSS, or things will get all messed up. So, in the case when there is no positioned parent element, it will be positioned related directly to the HTML element (the page itself). The position of the yellow box is now based on its parent position (the dotted border). Thus, moving 20px to the right and bottom from the top-left origin of the dotted border.
