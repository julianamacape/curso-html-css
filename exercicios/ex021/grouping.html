Grouping Tags

  • a tag <header> é o cabeçalho
  • a tag <main> é o conteúdo principal
  • a tag <footer> é o rodapé
  • dentro da tag <header> você pode colocar a tag <nav> (navigation links) que é comumente usada quando houver um bloco (block) significativo de links de navegação (ex.: menus)
  • dentro da tag <main> você pode colocar a tag section que é usada para agrupar elementos relacionados
  • dentro da tag <section> geralmente incluímos um título, a tag <article> e a tag <p>
  • dentro da tag <article> você pode colocar a tag <aside> que define uma seção que é tangencialmente relacionada ao conteúdo da página. Geralmente usada para publicidade, glossário, biografia do autor, informação de perfil ou links relacionados.
