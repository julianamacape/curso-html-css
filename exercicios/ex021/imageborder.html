Bordas com Imagens

Veja o código abaixo: 

    
        <style>
            h1 {
                border: 12px solid black;
                padding: 8px;
                border-image-source: url('pattern.jpg');
                border-image-slice: 10;
                border-image-repeat: repeat;
            }
        </style>
O 'border-image-repeat:' tem algumas opções:
  • repeat;
  • round;
  • space;
  • stretch;
É possível utilizar uma 'shorthand' para definir os atributos da borda com imagens. A ordem é:
  1. source
  2. slice
  3. repeate
    
        <style>
            h1 {
                border-image: url('filename.jpg') 12 stretch
            }
        </style>
