Modelo de Caixas (ou Box Model)

Todo elemento visível na tela é uma "caixa". E cada caixa tem sua anatomia.

Tipos de caixa

Block-level

  • sempre inicia numa linha nova
  • sempre quebra a linha quando acaba
  • sempre ocupa a largura total da tela

Inline-level

  • sempre inicia na mesma linha
  • não quebra a linha quando acaba
  • ocupa só o tamanho que precisa

Definindo dimensões da caixa

Ao definir os valores de "padding", "border" e "margin", você começa de cima (top) e vai no sentido horário definindo os demais lados (ou seja, segue para right, bottom e, por fim, left). A unidade de medida deve ser 'px', mas pode utilizar 'auto' o que significa que ficará centralizado.

Shorthand

Se tratando de "border" e "outline", a ordem correta é width style color. Exemplo:
border: 10px solid yellow;

Se tratando de "padding" e "margin", a ordem correta é top right bottom left. Exemplo:
padding: 30px auto 100px auto;

Forçando elementos

É possível forçar que um elemento "block-level" seja "inline-level" ou vice-versa. Exemplo: 

        
            <style>
                h1 {
                    display:inline
                }
            </style>
        
    
        
            <style>
                a {
                    display: block;
                }
            </style>
