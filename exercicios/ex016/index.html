Representando Cores em HTML

  • Por nome

    Esse texto é azul. 

    •                 
                    <p style="color: blue;">
                        Esse texto é azul.
                    </p>

  • Por Código Hexadecimal

    Esse texto é azul. 

                    
                    <p style="color: #0000ff;">
                        Esse texto é azul.
                    </p>

  • Por código RGB (Red / Green / Blue)

    Esse texto é azul. 

                    
                    <p style="color: rgb(0, 0, 255);">
                        Esse texto é azul.
                    </p>

  • Por código HSL (Hue / Saturation / Luminosity)

    Esse texto é azul. 

                    
                    <p style="color: hsl(240, 100%, 50%);">
                        Esse texto é azul.
                    </p>
