Testando gradientes (degradê) em CSS

Voltar para a página inicial.

Eu criei o degradê do plano de fundo dessa página da seguinte forma: 

        
            <style>
                body {
                    background-image: linear-gradient(to right, color01, color02);
                }
            </style>

  • Você coloca quantas cores quiser;
  • você direciona para onde vai o degradê
    1. para a direita "to right";
    2. para a esquerda "to left";
    3. para cima "to top"
    4. para baixo "to bottom"

O problema é que se você escolher a direção do degradê "to top" ou "to bottom" corre o risco de o background ficar meio "zebrado", como a imagem abaixo:

Nesse caso, faz-se necessária a seguinte configuração no seu código: 

        
            <style>
                * {
                    height: 100%;
                }
            </style>
