DaFont.com

  • acesse www.dafont.com
  • escolha a fonte que quiser
  • baixe o arquivo da fonte
  • lá no código, dentro da tag <style>, antes de incluir os seletores, insira a regra '@font-face'

Veja abaixo o exemplo do códido: 

    
        <style>
            @font-face {
                font-family: 'Marselle';
                src: url('mersella-font/Mersella.otf') format('opentype');
                src: url('mersella-font/Mersella.ttf') format('truetype');
            }
    
            h1 {
                font-family: 'Marselle', Arial, Helvetica, sans-serif;
            }
        </style>

Dentro de 'font-family', você dá o título que quiser à fonte baixada.
