Shorthand

Eventualmente você irá perceber que algumas configurações se repetem bastante. Por exemplo: 'font-family', 'font-size', 'font-weight' e 'font-style'.
Dá pra agrupar esse pessoal aí no que chamamos de shorthand.

A sequência correta para o uso de uma shorthand em fonte é (1) style (2) weight (3) size (4) family. Nada de usar vírgulas, hein? 

    
        <style>
            p {
                font: italic bold 3em Arial, sans-serif;
            }
        </style>
