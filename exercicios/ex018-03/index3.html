Tipografia (parte 2)

Tamanhos de Fontes

Recomenda-se seguir as orientações da W3School, ou seja, usar "px" e "em"

Medidas Absolutas

  • cm
  • mm
  • in
  • pt
  • px
  • pc
    
        <style>
            p {
                font-size:  2cm;
                            10mm;
                            3in;
                            18pt;
                            9px;
                            12pc;
            }
        </style>
pt e pc não são recomendadas para telas, somente para impressão.

Medidas Relativas

  • em
  • ex
  • rem
  • vw
  • vh
  • %

    
        <style>
            p {
                font-size:  1em;
                            1ex;
                            1rem;
                            20vw;
                            15vh;
                            40%;
            }
        </style>

O tamanho padrão de fonte é 16px = 1em

Peso, Estilo e Shorthand

Peso

Peso quer dizer a fonta mais larga ou mais estreita 

    
        <style>
            p {
                font-weight: lighter;
                             normal;
                             bold;
                             bolder;
                             200;
                             350;
                             etc
            }
        </style>

É possível usar numeração de 100 até 900 para definir o peso da fonte. Lembrando que nem todas as fontes respondem à configuração de font-weight.

Estilo

Importante lembrar que nem todas as fontes respondem à declaração de 'font-style' 

    
        <style>
            p {
                font-style: italic;
                            normal;
                            oblique;
                            etc...
            }
        </style>
